# Supermarket Day01 Final Cutout Pack

用途：
- 直接覆盖/合并到 Unity 项目根目录。
- 压缩包内目录从 `Assets/` 开始，文件名与当前代码绑定路径一致。
- Unity 打开项目后执行：`Supermarket -> Rebuild Art Catalog`。

包含素材：
- warehouse_corner.png
- pallet_box_stack.png
- fridge_double_drinks.png
- promo_stand_super_sale.png
- potted_plant_large.png
- floor_tile.png
- wall_tile.png
- mission_panel.png

注意：
- 不要改文件名。
- 不要改变目录层级。
- 解压目标应是项目根目录，而不是 `Assets/` 目录内部再套一层。
